2. 登录与注册

2.1 登录

  1. 输入账户名（从下拉列表选择或输入新用户名）
  2. 输入密码
  3. 点击"登录"按钮或按回车键

2.2 注册新账户

  1. 点击"注册"按钮
  2. 输入账户名（至少3个字符）
  3. 输入密码（至少6个字符）
  4. 确认密码
  5. 点击"注册"完成

2.3 切换账户

  • 菜单：管理账户 → 切换账户
  • 快捷键：Ctrl + L

3. 收支记录管理

3.1 添加记录

  1. 点击工具栏的"添加"按钮，或使用快捷键 Ctrl + N
  2. 在弹出的对话框中填写信息：
    • 类型：收入 / 支出
    • 日期：选择日期（默认今天）
    • 分类：选择或使用默认分类
    • 金额：输入金额
    • 支付方式：现金、微信、支付宝等
    • 经手人：选择家庭成员
    • 说明：可选，添加备注
  3. 点击"确定"保存

3.2 编辑记录

  1. 在表格中选中要编辑的记录（点击一行）
  2. 点击工具栏的"编辑"按钮，或双击该记录
  3. 修改信息后点击"确定"

3.3 删除记录

  1. 选中要删除的记录（可多选）
  2. 点击工具栏的"删除"按钮
  3. 确认删除
注意：删除的记录会移入回收站，可以在回收站中恢复。

3.4 筛选记录

类型筛选：

  • 在工具栏的类型下拉列表中选择：全部 / 收入 / 支出

日期筛选：

  1. 取消勾选"显示全部"
  2. 选择开始日期和结束日期
  3. 点击"筛选"按钮

3.5 分页功能

表格下方提供分页控件：

  • 首页：跳转到第一页
  • 上一页：跳转到上一页
  • 下一页：跳转到下一页
  • 末页：跳转到最后一页
  • 跳转：输入页码后按回车或点击跳转按钮

3.6 全选记录

使用快捷键 Ctrl + A 可以选中当前页的所有记录。

4. 统计分析

切换到"统计分析"标签页查看收支统计。

4.1 收支概览

页面顶部显示三个统计卡片：

  • 总收入：绿色卡片，显示总收入金额
  • 总支出：红色卡片，显示总支出金额
  • 结余：蓝色卡片，显示收支差额

4.2 日期筛选

  1. 选择开始日期和结束日期
  2. 点击"查询"按钮刷新统计数据

4.3 分类统计

使用"查看分类"下拉列表选择：

  • 支出分类：查看各支出类别的占比
  • 收入分类：查看各收入类别的占比

4.4 饼图

饼图展示各类别占总金额的百分比，不同颜色代表不同类别。

4.5 月度趋势

柱状图展示最近几个月的收支对比：

  • 绿色柱子代表收入
  • 红色柱子代表支出

5. 回收站

5.1 查看回收站

切换到"回收站"标签页。

5.2 恢复记录

  1. 选中要恢复的记录
  2. 点击"恢复"按钮
  3. 记录将被恢复到收支记录列表

5.3 永久删除

  1. 选中要永久删除的记录
  2. 点击"永久删除"按钮
  3. 确认后永久删除（不可恢复）

5.4 清空回收站

点击"清空回收站"按钮，清空所有已删除的记录。

6. 数据导入导出

切换到"导入导出"标签页。

6.1 导出数据

  1. 选择"导出数据"选项卡
  2. 选择导出格式：CSV 或 Excel
  3. 选择日期范围（可选）
  4. 点击"导出"按钮
  5. 选择保存位置

6.2 导入数据

  1. 选择"导入数据"选项卡
  2. 点击"选择文件导入"按钮
  3. 选择 CSV 或 Excel 文件
  4. 预览导入数据
  5. 点击"导入"按钮

6.3 支持的文件格式

  • CSV：逗号分隔值文件
  • Excel：.xlsx 格式（需要安装 openpyxl）

6.4 文件格式说明

7. 账户管理

7.1 修改密码

修改自己密码：

  • 菜单：管理账户 → 修改密码
  • 输入原密码、新密码、确认密码
  • 点击"确定"

管理员修改其他账户密码：

  • 菜单：管理账户 → 账户管理
  • 选择要修改密码的账户
  • 点击"修改密码"按钮

7.2 账户管理（仅管理员）

管理员可以：

  • 查看所有账户列表
  • 修改任意账户的密码
  • 删除非管理员账户

8. 主题设置

8.1 切换主题

在登录窗口顶部，点击"主题"下拉列表选择喜欢的主题：

  • 浅色：浅灰色背景
  • 灰色：中灰色背景
  • 蓝色：蓝色调背景
  • 绿色：绿色调背景
  • 紫色：紫色调背景
提示：选择的主题会自动保存到配置文件中，下次启动程序时会自动应用上次选择的主题。

9. 语言切换

9.1 切换语言

在登录窗口顶部，点击"语言"下拉列表选择语言：

  • 简体中文
  • 繁體中文
  • English
提示：选择的语言会自动保存到配置文件中，下次启动程序时会自动应用上次选择的语言。

常见问题

Q: 如何备份数据？

A: 使用"导入导出"功能中的"导出数据"功能，将数据导出为 CSV 或 Excel 文件。

Q: 误删了记录怎么办？

A: 可以在"回收站"中找到误删的记录，点击"恢复"按钮恢复。

Q: 如何添加新的分类？

A: 当前版本不支持自定义分类，使用预设的分类（工资、奖金、餐饮、交通等）。

Q: 如何更改主题？

A: 在登录窗口顶部的"主题"下拉列表中选择喜欢的主题。

Q: 如何切换语言？

A: 在登录窗口顶部的"语言"下拉列表中选择需要的语言。
